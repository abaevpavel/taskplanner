## Что это
HR назначает сотрудникам чеклисты из библиотеки, отмечает задачи в 3 состояниях (✓/×/N/A),
прикрепляет фото/заметки, экспортирует PDF. Иерархические задачи (`parent_id`),
drag-and-drop редактор шаблонов, AI-импорт чеклиста из картинки, синк сотрудников/вендоров
в Airtable через Make.com по расписанию (pg_cron).

**Сложность: средне-высокая.** Трудоёмкость даёт не CRUD, а: иерархический dnd-редактор
с undo, каскадная пропагация состояний, 3-state, PDF, AI-импорт.

## Простыми словами: кто что делает
- **HR-менеджер** — выбирает сотрудника, назначает ему чеклисты, отмечает галочки (✓ сделано /
  ✗ нет / N/A не нужно), цепляет фото и заметки, жмёт «скачать PDF».
- **Приложение (в браузере)** — показывает чеклист деревом (подзадачи под задачами),
  само считает прогресс, складывает все отметки и фото в базу, собирает PDF.
- **База (Supabase)** — хранит библиотеку чеклистов, сотрудников, кто что отметил, фото.
- **ИИ (через edge function)** — если HR загрузил картинку чеклиста, ИИ сам распознаёт пункты
  и строит из них дерево задач (не вбивать руками).
- **«Будильник» базы (cron) + Make** — отдельная мелкая функция: 2 раза в день сами обновляют
  контакты сотрудников/вендоров в Airtable (то же, что делает приложение HR-Sync Airtable).
> Главное: вся работа с чеклистами происходит **внутри приложения и его базы**; наружу
> (Make/Airtable) уходит только вспомогательный синк контактов.

## Scope (что строим)
- **2 экрана:** `/checklists` (каталог сотрудников + назначение чек-листов + прохождение пунктов +
  заметки/фото на двух уровнях + PDF), `/checklist/:id` (редактор шаблона: dnd-дерево, inline-edit,
  indent/nest, undo, дубль, side-nav, AI-импорт).
- **БД:** 8 таблиц (`checklists`, `checklist_items` self-FK + `links` jsonb + `photos[]` +
  `answer_options` jsonb, `employees`, `employee_types`, `employee_checklists` (+notes),
  `employee_checklist_progress` (completed/N-A/answer/notes/photos), `checklist_photos`,
  `employee_phase_preferences`) + констрейнты + RLS.
- **Storage:** 3 бакета (фото пунктов / фото назначений / production) + JPEG-компрессия.
- **Edge function:** `extract-checklist-from-image` (AI Gateway, парс дерева из картинки).
- **Auth/роли:** Admin + HR Manager. **БЕЗ Make/Airtable** — это приложение само хранит данные.
